# Pokedex React

## descrição
Aplicação web SPA que consome a PokeAPI para exibir dados de pokémons.

## tecnologias
- React
- Typescript
- Tailwind
- shadcn
- React Router
- Axios

## funcionalidades
- listagem de pokémons
- busca
- favoritos
- autenticação

## como rodar

git clone
npm install
npm run dev
